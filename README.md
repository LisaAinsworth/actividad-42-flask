## Descripción del proyecto

Este proyecto corresponde a la Actividad Evaluada 4.2 de Desarrollo de Software para Hardware.

La aplicación web fue desarrollada con Flask, HTML, CSS y Jinja en Ubuntu usando VirtualBox.  
El sistema muestra datos X-Y en una interfaz web con tres vistas conectadas entre sí.

## Objetivo

Representar datos X-Y mediante una página web construida con Flask, cumpliendo las restricciones solicitadas en la actividad:

- Sin uso de JavaScript.
- CSS escrito dentro del encabezado HTML.
- Uso de selectores CSS mediante id.
- Visualización personalizada de datos.
- Integración de tres vistas navegables.

## Vistas del proyecto

- Inicio: matriz 4x4 con la posición X-Y marcada.
- Vista 1: panel de coordenadas X e Y.
- Vista 2: tablero alternativo de estado X-Y.
